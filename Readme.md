简介：    
  GcwApi:这个python包下面放一些封装好的的接口和方法
  
     api：   封装好的接口，可以直接调用   
     core：  框架核心，封装了requests方法 
     
  GcwApiTest： 调用GcwApi中的接口和方法
  
     api_test: 单个接口测试
     data:     存放数据，如测试环境的地址
     libraries: 放一些测试库  
     reports： 存放测试报告 
     senario_test： 多个接口组成的基于场景的测试  
  
  
  
框架持续更新优化。。
